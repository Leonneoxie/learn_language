Python 2 and 3¶

Python2.7¶

print "hello"

hello

Python3.0¶

print "hello"

  File "tmp.py", line 1
    print "hello"
                ^
SyntaxError: invalid syntax

Python3.0¶

print("hello")

hello

Python2.7¶

print {}.keys()

[]

Python3.0¶

print({}.keys())

<dict_keys object at 0x385f50>

Python2.7¶

print None < None

False

Python3.0¶

print(None < None)

Traceback (most recent call last):
  File "tmp.py", line 1, in <module>
    print(None < None)
TypeError: unorderable types: NoneType() < NoneType()

Python2.7¶

print 1 / 2

0

Python3.0¶

print(1 / 2)

0.5
